# 💳 UPI Fraud Detection & Risk Analysis

## 📌 Project Overview

This project analyzes UPI (Unified Payments Interface) transaction data to identify fraud patterns, detect high-risk transactions, and generate actionable business insights. The analysis combines SQL, Python, and Power BI to perform data cleaning, exploratory analysis, and interactive dashboard creation.

The objective is to help financial institutions monitor fraudulent activities, reduce financial losses, and improve transaction security through data-driven decision-making.

---

## 🎯 Problem Statement

With the rapid growth of digital payments, fraudulent UPI transactions have increased significantly. Financial organizations require efficient methods to identify suspicious transactions, understand fraud trends, and minimize risks.

This project analyzes transaction data to uncover fraud patterns and provide meaningful insights using Business Intelligence tools.

---

## 🎯 Objectives

* Clean and prepare raw transaction data.
* Analyze transaction trends and customer behavior.
* Identify fraud-prone transaction types.
* Calculate fraud rates using SQL and Power BI.
* Build an interactive dashboard for business users.
* Generate business recommendations to reduce fraud.
